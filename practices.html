CONCEPTOS BÁSICOS EN PROGRAMACIÓN

¿Qué es una variable en programación?

Una variable es un espacio de almacenamiento para datos.

                
                    <var> miVariable = "Hola Mundo"; </var>

Cómo usar

Para usar una variable, simplemente la referenciamos por su nombre:

                
                    <script>
                        console.log(miVariable);
                    </script>

¿Qué es una constante en programación?

Una constante es un valor que no cambia después de su declaración.

                
                    <const> PI = 3.14; </const>

Cómo usar

Usar una constante es similar a usar una variable, pero no se puede reasignar:

                
                    <script>
                        console.log(PI);
                        PI = 3.1416; // Esto generará un error
                    </script>

Condicional if else

La estructura de control if-else ejecuta código dependiendo de una condición.

                
                    <if>
                        if (condición) {
                            // código si la condición es verdadera
                        } else {
                            // código si la condición es falsa
                        }
                    </if>

Cómo usar

Podemos cambiar el valor de la condición y observar cómo afecta el flujo del programa:

                
                    <script>
                        var edad = 18;

                        if (edad >= 18) {
                            console.log("Eres mayor de edad");
                        } else {
                            console.log("Eres menor de edad");
                        }
                    </script>

Switch case

La estructura de control switch selecciona entre múltiples casos.

                
                    <switch>
                        case valor1:
                            // código para valor1
                            break;
                        case valor2:
                            // código para valor2
                            break;
                        default:
                            // código por defecto
                    </switch>

Cómo usar

Podemos cambiar el valor y observar qué caso se ejecuta:

                
                    <script>
                        var diaDeLaSemana = "lunes";

                        switch(diaDeLaSemana) {
                            case "lunes":
                                console.log("Es lunes.");
                                break;
                            case "martes":
                                console.log("Es martes.");
                                break;
                            default:
                                console.log("No es ni lunes ni martes.");
                        }
                    </script>

Funciones

Una función es un bloque de código reutilizable que realiza una tarea específica.

                
                    <function> nombreFuncion(parametros) {
                        // código de la función
                    }
                    </function>

                    // Llamada a la función
                    nombreFuncion(argumentos);

Cómo usar

Definimos una función y luego la llamamos con los argumentos necesarios:

                
                    <script>
                        function saludar(nombre) {
                            console.log("Hola, " + nombre + "!");
                        }

                        saludar("Juan");
                    </script>
